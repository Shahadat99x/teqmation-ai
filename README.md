# ClientFlow AI

ClientFlow AI is a workflow automation SaaS for small education consultancies.

## Project Structure

- `apps/web`: Next.js dashboard and public forms.
- `apps/api`: FastAPI backend for business logic.
- `packages/ui`: Shared UI components.
- `packages/types`: Shared TypeScript types.
- `packages/config`: Shared ESLint, Prettier, and Tailwind configurations.
- `infra/n8n`: Workflow automation configurations.
- `supabase/migrations`: Database schema and migrations.
- `docs/`: Project documentation and session memory.
